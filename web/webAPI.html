Universal API to LOD
Automatically generating SPARQL queries to access LOD endpoints


Info

The Universal API can be queried by using this website or programmatically pointing at UniversalAPIQuery and using the paremeters 'endpoint' and 'query'.

An example of using the API directly will be the following: https://wake.dlsi.ua.es/UniversalAPIQuery?endpoint=http://bio2rdf.org/sparql&query=SELECT+%3Fsubject+%3Fpredicate+%3Fobject%0D%0AWHERE+%7B%3Fsubject+%3Fpredicate+%3Fobject%7D+%0D%0ALIMIT+100

More details at GitHub page.
